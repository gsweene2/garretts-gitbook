pyenv basics, changing versions

List Current Version

$ pyenv versions
  system
  3.6.11
* 3.7.8 (set by /Users/garrettsweeney/.pyenv/version)

Install new versions

$ pyenv install 3.8.0

Switch Version

$ pyenv global 3.8.0
$ pyenv versions
  system
  3.6.11
  3.7.8
* 3.8.0 (set by /Users/garrettsweeney/.pyenv/version)

$ python -V
Python 3.8.0
